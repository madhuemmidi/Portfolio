# Madhuri's Portfolio

Simple static portfolio site — just HTML, CSS, and a tiny bit of JS. No build step.

## Run locally in VS Code

1. Open this folder in VS Code (`File > Open Folder...`).
2. Install the **Live Server** extension (by Ritwick Dey).
3. Right-click `index.html` → **Open with Live Server**.

Or just double-click `index.html` to open it in your browser.

## Publish (free)

### GitHub Pages
1. Create a new repo on github.com (e.g. `portfolio`).
2. Upload these files.
3. Repo → **Settings → Pages → Source: `main` branch / root** → Save.
4. Live at `https://<your-username>.github.io/portfolio/` in ~1 min.

### Netlify Drop
1. Go to https://app.netlify.com/drop
2. Drag this folder onto the page → instant URL.
